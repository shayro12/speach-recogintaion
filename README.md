# Speech Recognition

This repository demonstrates the usage of the speech recognition module in Python.

## Prerequisites

Make sure you have Docker installed on your server before proceeding with the following steps.

## Instructions

1. Build the Docker image:

`docker build -t flask-app .`

2. Run the Docker container and expose the application:

`docker run -p 5000:5000 flask-app:latest`

These commands will set up and run the speech recognition application on your server.
